Chamando a API no Angular

Para consumir a API no Angular, basta utilizar o serviço HttpClient, concatenando os parâmetros necessários conforme o exemplo de código abaixo.

  • Os parâmetros obrigatórios são:
  • link: o ID da sua planilha do Google;
  • pagina: o nome da aba onde os dados estão armazenados;
  • celulas: o intervalo (range) das células que deseja retornar.
  • Com esses parâmetros, os dados serão carregados corretamente na sua aplicação.
